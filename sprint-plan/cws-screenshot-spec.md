## Pre-Capture Setup Checklist

Before taking any screenshot:

- [ ] Use a real ASIN — **primary: B0DZDBQD4K**. Avoid placeholder/mock data — reviewers will notice.
- [ ] Confirm the extension is in **dark theme** (default appearance)
- [ ] Chrome window: clean profile — no other extension icons in toolbar, bookmarks bar hidden
- [ ] Side panel width: **~400px** (standard — drag to exact width before starting)
- [ ] Browser zoom: **100%** — no scaling artifacts
- [ ] Capture at **1280×800** — position Chrome window to this exact size (`Ctrl+Shift+I → ⋮ → Settings → Dimensions`)
- [ ] **Blur** any email address visible in the top-right account area
- [ ] Verify legibility by scaling preview to 400px wide in an image editor before finalizing

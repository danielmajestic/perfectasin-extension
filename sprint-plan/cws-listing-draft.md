## PRIVACY POLICY OUTLINE

**Required URL:** Must be live at `perfectasin.com/privacy` before CWS submission.

### What We Collect

| Data Type | Purpose | Stored Where | Retention |
|---|---|---|---|
| Google Account (email, display name, UID) | Authentication and account identification | Firebase Auth | Until account deleted |
| Analysis usage count | Enforce free tier limits | Firestore (per user UID) | Until account deleted |
| Amazon page data (ASIN, title, bullets, description, price, category) | Pass to Claude API for analysis | Not stored — passed in real-time and discarded | Never persisted |
| Claude API responses (scores, variations, ICP) | Displayed to user in extension | `chrome.storage.session` (in-memory only) | Cleared on browser close |
| Stripe customer ID + subscription status | Billing and tier enforcement | Firestore | Until account deleted |

### What Goes to the Claude API

When you run an analysis, the following product data is sent to Anthropic's Claude API:
- Product title
- Bullet points (if analyzing bullets)
- Product description (if analyzing description)
- Category, brand, price
- ASIN identifier

**What is NOT sent to Claude API:** Your name, email, Google account details, or any personal information. Only product listing data is transmitted.

Anthropic's data handling: Data sent to the Claude API is not used to train models (per Anthropic's API Terms of Service, as of 2026). Review Anthropic's privacy policy at anthropic.com/privacy.

### What We Do NOT Collect

- Browsing history outside of amazon.com product pages
- Payment card details (processed directly by Stripe — never touches our servers)
- Your Amazon seller account credentials
- Any data from non-Amazon pages

### Third-Party Services

| Service | Purpose | Their Privacy Policy |
|---|---|---|
| Firebase (Google) | Authentication, database | firebase.google.com/support/privacy |
| Anthropic Claude API | AI analysis | anthropic.com/privacy |
| Stripe | Payment processing | stripe.com/privacy |
| Google Cloud Run | API hosting | cloud.google.com/terms/cloud-privacy-notice |

### Data Rights

Users may request deletion of their account and all associated data by emailing support@perfectasin.com. Account deletion removes: Firebase auth record, Firestore usage data, Stripe customer data. Deletion completed within 30 days.
